Express + React 搭建简易web微博

Mar 2, 2017

最近用 React 写了个简单的web微博，因为js的跨域问题，无法直接请求web Weibo的API，所以只能使用Node来中转一下，这样玩法也更丰富，随便处理数据。目前的功能只包括看微博和查看大图。

微博的自动登录使用的是nodejs-weibo-login,但是这个库是为了pc端微博设计的，最初还是可以兼容web版的微博登录，后来估计是新浪修改了规则，不再兼容了，所以目前只能手动传 cookie 了。
基本没用第三方的react类库，就是个很简单的东西。
